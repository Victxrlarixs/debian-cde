# Netscape Navigator User Guide

Experience web browsing from 1994 with Netscape Navigator 1.0!

## Opening Netscape

- Click the Netscape icon in the Front Panel
- Use keyboard shortcut: `Ctrl+Alt+N`
- Open from App Manager

## Interface

### Components

1. **Title Bar** - "Netscape Navigator"
2. **Menu Bar** - File, Edit, View, Go, Help
3. **Toolbar** - Navigation buttons
4. **Address Bar** - URL input
5. **Content Area** - Web page display
6. **Status Bar** - Loading status

### Toolbar Buttons

- **Back** - Previous page
- **Forward** - Next page
- **Home** - Go to home page
- **Reload** - Refresh current page
- **Stop** - Stop loading

## Navigation

### Visiting Pages

1. Click the address bar
2. Type URL
3. Press Enter

### Using Buttons

- Click **Back** to go to previous page
- Click **Forward** to go to next page
- Click **Home** to return to start page

### Keyboard Shortcuts

| Shortcut          | Action            |
| ----------------- | ----------------- |
| `Ctrl+L`          | Focus address bar |
| `Enter`           | Navigate to URL   |
| `Backspace`       | Go back           |
| `Shift+Backspace` | Go forward        |
| `Ctrl+R`          | Reload page       |
| `Ctrl+H`          | Go to home        |
| `Esc`             | Stop loading      |

## Differences from Modern Browsers

Netscape Navigator 1.0 is a recreation for nostalgia:

- Limited to internal pages
- No external internet access
- Authentic 1994 interface
- Classic navigation buttons

## Troubleshooting

### Page Won't Load

Internal pages should load instantly. If not, try reloading.

### Back Button Disabled

You're on the first page. Navigate forward first.

### Address Bar Not Working

Make sure you press Enter after typing.

## Fun Facts

### History

- Netscape Navigator was released in 1994
- It dominated the early web
- Introduced many web standards
- Competed with Internet Explorer
- Open-sourced as Mozilla in 1998

### Legacy

- Influenced modern browsers
- Pioneered web navigation
- Introduced SSL/HTTPS
- Created JavaScript
- Shaped the internet
