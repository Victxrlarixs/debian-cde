# File Manager User Guide

The File Manager lets you browse, organize, and manage files in the virtual filesystem.

## Opening File Manager

- Click the File Manager icon in the Front Panel
- Use keyboard shortcut: `Ctrl+Alt+F`
- Double-click a folder on the desktop

## Interface Overview

### Main Components

1. **Navigation Bar** - Current path and navigation buttons
2. **Toolbar** - Quick action buttons
3. **File List** - Files and folders
4. **Status Bar** - Selected items and info

### View Modes

**Icon View**

- Large icons for files and folders
- Good for visual browsing
- Shows file type icons

**List View**

- Detailed file information
- Name, size, date modified
- Good for many files

**Tree View**

- Hierarchical folder structure
- Expand/collapse folders
- Navigate quickly

## Navigation

### Basic Navigation

```
Home → Documents → Projects → myproject
```

- **Forward/Back** - Navigate history
- **Up** - Go to parent folder
- **Home** - Go to home directory
- **Path Bar** - Click any folder in path

### Keyboard Navigation

| Shortcut    | Action               |
| ----------- | -------------------- |
| `Enter`     | Open selected item   |
| `Backspace` | Go to parent folder  |
| `Alt+↑`     | Go up one level      |
| `Alt+Home`  | Go to home directory |
| `Alt+←`     | Go back              |
| `Alt+→`     | Go forward           |

## File Operations

### Creating Files and Folders

**New Folder**

1. Click "New Folder" button or press `Ctrl+N`
2. Enter folder name
3. Press Enter

**New File**

1. Right-click in empty space
2. Select "New File"
3. Enter filename

### Copying and Moving

**Copy**

1. Select file(s)
2. Press `Ctrl+C` or right-click → Copy
3. Navigate to destination
4. Press `Ctrl+V` or right-click → Paste

**Move**

1. Select file(s)
2. Press `Ctrl+X` or right-click → Cut
3. Navigate to destination
4. Press `Ctrl+V` or right-click → Paste

**Drag and Drop**

- Drag file to another folder to move
- Hold `Ctrl` while dragging to copy

### Renaming

1. Select file
2. Press `F2` or right-click → Rename
3. Type new name
4. Press Enter

### Deleting

1. Select file(s)
2. Press `Delete` or right-click → Delete
3. Confirm deletion

## Selection

### Selecting Files

- **Single**: Click file
- **Multiple**: Hold `Ctrl` and click files
- **Range**: Click first, hold `Shift`, click last
- **All**: Press `Ctrl+A`

### Selection Shortcuts

| Shortcut      | Action           |
| ------------- | ---------------- |
| `Ctrl+A`      | Select all       |
| `Ctrl+Click`  | Add to selection |
| `Shift+Click` | Select range     |
| `Ctrl+I`      | Invert selection |

## Context Menu

Right-click on files or folders for options:

- **Open** - Open file/folder
- **Open With** - Choose application
- **Cut** - Move file
- **Copy** - Copy file
- **Paste** - Paste copied items
- **Delete** - Remove file
- **Rename** - Change name
- **Properties** - View details

## Search

### Finding Files

1. Press `Ctrl+F` or click Search button
2. Enter search term
3. Choose search location
4. View results

### Search Options

- **Name** - Search by filename
- **Content** - Search file contents (text files)
- **Type** - Filter by file type
- **Date** - Filter by modification date

## File Properties

View detailed information:

1. Right-click file
2. Select "Properties"
3. See:
   - Name
   - Type
   - Size
   - Created date
   - Modified date
   - Path

## Tips & Tricks

### Quick Navigation

- Type first letter of filename to jump to it
- Use breadcrumb path for quick navigation
- Bookmark frequently used folders

### Organizing Files

- Create folders for different projects
- Use descriptive names
- Keep similar files together
- Delete unused files regularly

### Keyboard Efficiency

- Learn the essential shortcuts
- Use `Ctrl+A` to select all
- Use `F2` for quick rename
- Use `Backspace` to go up

## Troubleshooting

### Can't Find a File

- Use the search function
- Check all folders
- Look in home directory

### Can't Delete a File

- Make sure file isn't open
- Check if you have permission
- Try closing all applications

### File Won't Open

- Check file type
- Try "Open With" and choose application
- File might be corrupted
