# Tips & Tricks

Advanced tips and hidden features to master CDE Time Capsule.

## Productivity Hacks

### Master Keyboard Shortcuts

Learn these power-user shortcuts:

```
Ctrl+Alt+1-4    Switch workspaces instantly
Alt+Tab         Cycle through windows
Alt+F4          Close window quickly
Ctrl+Alt+D      Show desktop
```

### Use Workspaces Strategically

- Workspace 1: Focus mode (one app, full screen)
- Workspace 2: Multitasking (multiple windows)
- Workspace 3: Reference materials
- Workspace 4: Background tasks

### Window Management

- Double-click title bar to maximize
- Drag to edge to snap (if implemented)
- Use Alt+Space for window menu
- Minimize unused windows
